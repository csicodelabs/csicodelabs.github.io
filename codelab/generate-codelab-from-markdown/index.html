Codelab是一种以动手实操为主的学习方式，中软国际MNC能力中心今后将会大量采用这种形式来对项目团队进行赋能。在Codelab模式下，讲师将要负责输出课题、编写Codelab内容和进行少量的课程指导，受训人员则需要更加主动地完成课题。

至于导师如何『输出课题、编写Codelab内容』则是本课程会带给你的。

前置条件

  • 你要有一个文本编辑器，推荐Visual Studio Code，因为它提供实时预览
  • claat工具 从这里下载

可以在这里看看GitHub的markdown格式指导，https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet

Codelab的文件格式可以看这里。

首先你需要填充一些元数据，包括简介和作者还有id。可以直接复制这个文件，然后改动，https://github.com/tomtom-gangdam-style-for-map/tomtom-gangdam-style-for-map.github.io/blob/master/gangdam.md

命令如下

 claat export generate-codelab-from-markdown.md

生成文件夹，名字和文件中元数据id所指定的值一致

 claat serve generate-codelab-from-markdown.md

在http://localhost:9090 启动服务，点击刚才生成的文件夹即可预览。
